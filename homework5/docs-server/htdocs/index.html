üìã Available Endpoints

GET

/api/health

Health check - Returns API and database status

GET

/api/spots

Get DX spots with optional filtering and pagination

Parameters: limit, offset, band, frequency_min, frequency_max, dx_call, spotter_call, mode, since, until

GET

/api/spots/recent

Get the most recent DX spots from the last 24 hours

Parameters: limit, hours

GET

/api/bands

Get activity statistics for all amateur radio bands

GET

/api/stats

Get comprehensive database statistics

GET

/api/callsigns/top

Get the most active DX stations and spotters

Parameters: limit, hours

GET

/api/frequency/histogram

Get frequency distribution data

Parameters: bins, band, hours

GET

/api/activity/hourly

Get spot activity broken down by hour

Parameters: hours, timezone
