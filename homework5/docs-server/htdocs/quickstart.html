‚ö° DX Cluster API - Quick Start

Get started in 5 minutes

Step 1: Check API Health

Verify the API is running:

curl http://localhost:8080/api/health

Expected Response:

{"status":"healthy","database":"connected"}

Step 2: View API Info

Get an overview of all available endpoints:

curl http://localhost:8080/api

Step 3: Get Recent Spots

Fetch the 5 most recent DX spots:

curl http://localhost:8080/api/spots/recent?limit=5

Step 4: Filter by Band

Get 20m band activity:

curl "http://localhost:8080/api/spots?band=20m&limit=10"

Step 5: View Statistics

Get comprehensive database statistics:

curl http://localhost:8080/api/stats

üí° Tips & Tricks

Pretty Print JSON Output

curl "http://localhost:8080/api/spots/recent" | jq '.'

Save Results to File

curl "http://localhost:8080/api/spots/recent?limit=50" > spots.json

Real-time Monitoring Script

while true; do
  clear
  curl -s "http://localhost:8080/api/spots/recent?limit=5" | jq '.spots[] | "\(.dx_call) on \(.frequency) (\(.band))"'
  sleep 30
done

Export to CSV

curl -s "http://localhost:8080/api/spots/recent?limit=100" | \
  jq -r '.spots[] | [.dx_call, .frequency, .band, .spotter_call] | @csv' > spots.csv
